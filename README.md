## Cómo instalar esta versión

1. Descomprime el ZIP.
2. Copia todos los archivos.
3. En GitHub Desktop: `Repository → Show in Explorer`.
4. Pega los archivos dentro de la carpeta del repositorio.
5. Acepta **Reemplazar los archivos en el destino**.
6. En GitHub Desktop escribe: `Publicar versión final 1.0`.
7. Pulsa `Commit to main`.
8. Pulsa `Push origin`.
9. Espera uno o dos minutos y recarga la web con `Ctrl + F5`.
