# Glow by Antho Studio conectado a Supabase

## Archivos
- `index.html`: página pública para Instagram.
- `admin.html`: acceso privado de Anthonia.
- `config.js`: conexión pública con Supabase y WhatsApp.

## Publicación rápida
1. Abre tu repositorio `glowbyantho` en GitHub.
2. Entra a **Code**.
3. Pulsa **Add file → Upload files**.
4. Sube todos estos archivos y la carpeta `assets`.
5. Pulsa **Commit changes**.
6. En Vercel, crea una cuenta con GitHub.
7. Pulsa **Add New → Project** e importa `glowbyantho`.
8. Framework: **Other**. No agregues comandos de compilación.
9. Pulsa **Deploy**.

El enlace público será similar a `https://glowbyantho.vercel.app`.
El panel privado será `https://glowbyantho.vercel.app/admin.html`.

## Seguridad
Nunca publiques ni compartas la Secret key, service_role key ni contraseña de la base de datos.
